Bootstrap es un popular y gratuito framework front-end de código abierto que proporciona un conjunto de herramientas de HTML, CSS y JavaScript listas para usar, como componentes de interfaz (botones, menús, formularios) y un sistema de diseño de cuadrícula responsivo, para que los desarrolladores puedan crear rápidamente sitios y aplicaciones web adaptables a diferentes tamaños de pantalla y dispositivos.

Diseño responsivo: Su sistema de cuadrícula (grid system) permite crear diseños que se ajustan automáticamente a diferentes tamaños de pantalla, desde móviles hasta ordenadores de escritorio.
